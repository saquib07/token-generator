How to deploy React app on github:

1. npm install gh-pages --save-dev
2. then crete the github repository and follow the instructions there
3. "homepage": "http://<github-username>.github.io/<github-repository-name>", 
4. "predeploy": "npm run build",
5. "deploy": "gh-pages -d build",
6. npm run deploy
